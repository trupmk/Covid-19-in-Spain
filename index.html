About

The first case of COVID-19 in Spain was reported 118 days ago on 2/1/2020. Since then, the country has reported 238,564 cases, and 27,121 deaths. This website is all about the analysis to visualize the number of new COVID-19 cases, deaths, the PCR tests and everything.
